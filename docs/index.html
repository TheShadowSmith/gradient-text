Gradient Text Utility

A little CSS utility I created based on a text color effect I found while surfing the Stadia Developer Website.

How to use it

  1. Swipe the gradient-text CSS class from the gradient-text.css file.
  2. Modify the CSS gradient HEX values with whatever colors you wish to be in your gradient and change the angle of the gradient to whatever you prefer.
  3. Add the gradient-text CSS class to the elements that contain text you wish to apply the effect to.

A super simple utility created by Shadow Smith.
